OHBM API
========

This is a wrapper for the OHBM API, for which the original spec and code is available at the `ohbm <https://github.com/vsoch/ohbm>`_, repo. This API is not under development, not intended for public use, and required authentication. If you are a developer or hacker, please submit issues and requests to the `issue board <https://github.com/vsoch/ohbm/issues>`_.


Under Development
'''''''''''''''''

The tool currently is a wrapped for functions related to the following for the OHBM conference:

- Abstracts
- Attendees
- Authenticate
- Categories
- Events
- Exhibitors
- Planner
- Roomset
- System
